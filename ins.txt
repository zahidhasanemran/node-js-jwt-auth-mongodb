01. npm init -y
02. npm install express mongoose body-parser cors jsonwebtoken bcryptjs --save


==> Express is for building the Rest apis
==> Mongoose provides a straight-forward, schema-based solution to model your application data. It includes built-in type casting, validation, query building, business logic hooks and more, out of the box.
==> body-parser helps to parse the request and create the req.body object
==> cors provides Express middleware to enable CORS
==> Bcrypt is a secured way to store passwords in my database regardless of whatever language my app’s backend is built in — PHP, Ruby, Python, Node.js, etc.
